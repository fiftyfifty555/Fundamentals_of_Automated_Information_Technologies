# Database Design for an Online Educational Platform

A database design project for an online educational platform similar to Stepik.  
The system models the full cycle of interaction between users and educational content, including user registration, course enrollment, lessons, steps, grades, achievements, comments, promo codes, payments, and role-based access.

The project includes conceptual, logical, and physical database design, ER modeling, relational schema development, SQL query implementation, database size estimation, and analytical reporting.

A PostgreSQL database was designed and populated using ETL processes. Synthetic test data was generated with Python and Faker, transformed according to database constraints and business rules, and loaded into the database using batch inserts with transaction handling.

The work also includes SQL-based analytics for course popularity, user activity, course ratings, promo code efficiency, retention rate, inactive users, and attachment distribution across courses.
